Conway's Game of Life

This is the classic Conway's Game of Life with some tweak to it. Conway's Game of Life is a cellular automaton that was created by John Horton Conway. This is a 0 player game, in which the only determinant of how the evolution process takes place is the given initial state. This project was completed for a Software Design course in C++ and contained several adjustments to the original game.

When creating this project, it was essential to create header files, classes, vectors, and diligent use of namespaces. There was a large amount of files involved, thus, this project taught me how to avoid issues with overlapping namespaces. It also familiarized me with the concepts like vectors of vectors, file hierarchy within a project, and how to complete header files.

This project begins by reading from a text file that provides the inital input of the game. Once the game has collected the initial state, it will ask for the number of generations that you would like to compute. The program will then proceed in computing the generations using the conditions and rules of the game. Upon completion, the program will display the initial state, intermediate states, the final state to the console and output a text file containing the shape of every state.
